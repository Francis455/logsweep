LogSweep

A Rust CLI Tool for Cleaning Log Files

About LogSweep

LogSweep is a command-line tool built in Rust to help you clean and manage log files efficiently. It supports automatic cleaning and archiving of logs.

How to Use

Open a terminal and run:

cargo run -- [options] log_file.log

Features

  • Automatic cleaning of log files
  • Archiving old logs
  • Fast and efficient Rust performance
